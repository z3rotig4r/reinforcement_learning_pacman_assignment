# Reinforcement Learning Pacman Assignment

UC Berkeley AI Pacman Project - Reinforcement Learning

## Projects

### Project 1: Search & Multi-Agent
- Search algorithms (DFS, BFS, UCS, A*)
- Multi-agent search (Minimax, Alpha-Beta, Expectimax)
- Evaluation functions

### Project 2: Reinforcement Learning
- Value Iteration (MDP)
- Q-Learning
- Approximate Q-Learning

## Usage

```bash
# Project 1
cd project1
python pacman.py

# Project 2
cd project2
python pacman.py
python gridworld.py
```
